# VGTab Project — Copilot Memory

## Project Overview

- **App:** VGTab (VegaTab) — macOS app for generating AMD Vega Soft PowerPlay Tables
- **Location:** `/Applications/VGTab.app/` (original), `~/VGTab-git/` (git repo)
- **GitHub:** <https://github.com/yoy123/VegaTab>
- **Git user:** yoy123 / <danlambiase@ymail.com>

## Supported GPUs

Vega 64, Vega 56, Vega FE, Vega FE 8G, Vega 64 Liquid Cooling, Vega FE Liquid Cooling, WX 9100, WX 8200

### Tuning Rationale

- P4-P5 voltage flat at 1000mV — reduces power/heat in mid-range
- P7 clock at 1660 MHz — above stock for extra headroom
- P3 memory at 1050 MHz — above default 945 MHz
- TDC at 42% — below default 50%
- Target temp 70°C, fan target 3000 RPM

## Architecture

- AppleScript applet wrapping HTML/JS GUI via `vgtab://` URL scheme
- `VGtab.sh` decodes base64 settings → hex → binary PowerPlay table
- Binary injected into kext plist via `PlistBuddy`
- Output: `VegaTab_<GPU>.kext` on Desktop
- Chart.js for interactive visualization
- Settings files: plain text, one key=value per line
